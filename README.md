# 2048 數學課（颱風雨備・線上限定）

數感實驗室｜用益智遊戲 **2048** 帶中高年級認識指數成長、次方、思考策略，以及 AI 怎麼玩遊戲的三節線上課簡報。

## 三節內容
1. **認識 2048 與成長的祕密** — 玩 2048、指數 vs 線性、印度米粒傳說（一碗飯 → 全台灣一年的米）。
2. **倍數的力量** — 世界紀錄 65536、次方（連乘）表示法、高手訣竅（最大磚釘角落＋由大到小排成樓梯）。
3. **2048 大變身與 AI** — 費波那契數列版、立體 3D 版、貪食蛇版，以及現場看 AI 玩 2048。
